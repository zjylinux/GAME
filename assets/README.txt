  废土/丛林幸存者 · 2.5D 街机射击  — 资源说明

【当前状态】游戏已改为 100% 纯代码手绘渲染：
- 角色（玩家、野人、Boss）全部用 Canvas 路径/渐变绘制
- 背景（蓝天丛林、泥土路、棕榈树、物件）全部代码绘制
- 音效用 WebAudio 合成，无需音频文件

因此 assets/ 目前【不需要任何图片素材】。
manifest.json 仅保留为占位，供未来想换回图片贴图时使用。

如未来要重新接入图片贴图：
1) 把 PNG 放进 assets/
2) 在 manifest.json 里登记条目，格式：
   { "名字": { "src": "文件名.png", "frames": 帧数 } }
3) game.js 的 loadManifest() 会自动加载；缺失则自动回退手绘
